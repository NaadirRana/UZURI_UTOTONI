# Uzuri Utotoni: Web-Based Centralized Digital Infant Healthcare System

A modern, responsive healthcare application designed to track and identify newborns in Kenyan public hospitals. This system addresses data fragmentation and manual overhead by digitalizing infant registration, centralized observation monitoring, feeding cycles, and immunization schedules.

## 🚀 Collaborative Local Deployment Instructions
To run this application locally inside a unified XAMPP architecture setup:
1. Open Git Terminal inside your local machine web host path: `cd C:/xampp/htdocs/`
2. Clone this application directory: `git clone <YOUR_GITHUB_REPOSITORY_URL>`
3. Launch XAMPP Control Panel and start the Apache web server and MySQL database engines.
4. Execute the system database setup schema directly inside local `phpMyAdmin`.
5. Access the application workspace via your browser at `http://localhost/YOUR_REPO_NAME/dashboard.html`.
